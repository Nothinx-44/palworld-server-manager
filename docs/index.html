Frequently Asked

Does Palworld ban you for this?

No. This is an admin tool like editing config files. Palworld's REST API is intentionally public.

What happens if my PC crashes?

The server is a Windows service. It auto-restarts on reboot without manual intervention.

Do I need technical knowledge?

No. One-click installer guides you step-by-step. No command line needed.

Is it really free?

100% free and open source (MIT license). Your only cost is electricity to run your PC.

Can I move my server later?

Yes. Backups are standard Palworld saves—compatible with any server.

Does it track my data?

No. Zero telemetry. Everything stays on your machine. Fully auditable open source code.
